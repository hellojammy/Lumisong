这张三维声音图谱是什么？

这是把一段任意音频（音乐、语音、鸟鸣或环境声）变成的三维声音图谱——每颗球 = 一个可视化的声音事件。

  • 位置：横向 = 时间顺序，纵向 = 频率高低（音色亮度），纵深 = 音调纯净度——挨得近的球，时间或音色就相近；聚成一簇 = 重复相似的乐句、节奏或音色动机
  • 颜色：频谱带宽（音色的「宽窄」，对照左侧色条）
  • 大小：响度——声音越强球越大
  • 形状：光滑球 = 纯音（似口哨），多面晶体 = 沙哑宽频音
  • 连线：按先后顺序串起每个声音事件的轨迹——已播放的是实线，还没播放到的是虚线

播放时跳来跳去的亮光：哪颗球爆亮，就是此刻正在响的那个声音事件；白色光点拖着尾迹在球间穿梭，是「声音」沿着时间顺序飞行的轨迹。

核心视觉语言最初受 Lucio Arese 的《Seeing Birdsong》启发；本项目已定位为面向音乐、语音、鸟鸣与环境声的独立三维声音图谱。
