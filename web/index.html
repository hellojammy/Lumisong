如何使用 Lumisong

Lumisong（鸣光）把一段任意音频（音乐、语音、鸟鸣或环境声）变成三维声音图谱——每个发光体对应一个检测到的声音事件。

怎么读画面

  • 位置：横向是时间，纵向是音色亮度，纵深是音调纯净度；靠近的光体，时间或音色往往相近
  • 颜色：频谱宽窄（对照左侧色条）
  • 大小：响度——越强越大
  • 连线：按时间顺序串联轨迹；已播放为实线，未播放为虚线

播放时

最亮的那颗是当前正在响的事件；流光沿轨迹穿梭，表示播放进度。

底栏控制

  • 配色 / 形态：切换色谱与几何（玻璃球、光针、涟漪、晶钻、星环）——同一数据，不同外观
  • 运镜：智能运镜（默认）、匀速、自由、飞船驾驶；「更多」含飞船穿梭、呼吸环绕
  • 上传 / 录音：分析你自己的音频并可视化
  • 设置：读图标注（色条与频率数字）、云雾、连击、渐隐谢幕、余韵呼吸

视觉语言受 Lucio Arese《Seeing Birdsong》启发；本项目为面向通用音频的独立实现。
