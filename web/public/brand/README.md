# Lumisong Brand Assets

Canonical logo direction: **glowing sound-event atlas**.

Use these assets for the project unless a later brand spec explicitly supersedes them:

- `lumisong-mark.svg` — canonical full-color mark / app icon source.
- `lumisong-app-icon.svg` — app-icon wrapper for icon export.
- `lumisong-flat-mark.svg` — simplified single-color constrained-use mark.
- `lumisong-lockup.svg` — full lockup with `Lumisong` primary and `鸣光` secondary.
- `github-readme-hero.svg` / `github-readme-hero.png` — primary GitHub README banner.
- `github-social-card.png` — GitHub/social preview card.
- `reference/lumisong-logo-final-image2-board.png` — final Image 2 visual reference board.

Generated platform outputs:

- Web: `/favicon.svg`, `/favicon-16x16.png`, `/favicon-32x32.png`, `/apple-touch-icon.png`, `/icon-192.png`, `/icon-512.png`, `/maskable-icon-512.png`, `/site.webmanifest`.
- iOS: `ios/Lumisong/Assets.xcassets/AppIcon.appiconset/`.
- macOS: `macos/Resources/LumisongIcon.icns`.

Do not replace the mark with birds, music notes, microphones, waveform bars, records, or headphone icons. The logo represents many audio events in 3D space, traversed by one playback light path.
